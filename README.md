# Goblin Game

Игра, в которой нужно ловить гоблина, появляющегося в случайных ячейках.

## Особенности

- Custom-курсор в виде молотка
- Гоблин появляется на 1 секунду в случайной ячейке
- При клике на гоблина начисляется +1 балл
- При промахе (клик по пустой ячейке или пропуск появления) увеличивается счётчик промахов
- После 5 промахов игра завершается
